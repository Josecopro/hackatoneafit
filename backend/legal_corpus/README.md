# Corpus Legal Local (Modo sin vectorial)

Este directorio se usa para el flujo mínimo de agentes sin base vectorial.

## Qué archivos poner aquí

- `ley_1755_2015.md`: texto oficial de la Ley 1755 de 2015.
- `decreto_833_2015.md`: texto oficial del Decreto 833 de 2015.
- `constitucion_articulo_23.md`: artículo 23 (y relacionados si aplica).
- `secretaria_desarrollo_economico_competencias.md`: funciones y alcance de la Secretaría.

## Recomendación de contenido

- Guarda el texto completo de cada norma (no solo fragmentos).
- Puedes incluir al inicio un resumen breve (5-10 líneas).
- El agente hará extracción de fragmentos relevantes de forma determinista.

## Fuentes permitidas (.gov.co)

Para este flujo mínimo, limita URLs oficiales a páginas de la Secretaría:

- https://www.medellin.gov.co/es/secretaria-desarrollo-economico/
- https://www.medellin.gov.co/es/secretaria-desarrollo-economico/que-hace-la-secretaria/

No se deben usar fuentes de otras secretarías para responder automáticamente.
